# AI Renaissance

> 多智能体投资决策引擎 - 让AI时代的财富平权成为可能

## 项目愿景

通过多智能体架构，将机构级的投研能力封装成可复用的基础设施。

**核心价值**：不是给你一个买卖信号，而是让你理解为什么这笔交易值得做。

## 架构概览

```
┌─────────────────────────────────────────────────────────┐
│                    认知层（推理链输出）                      │
│           "为什么这笔交易值得做" ← 每个决策都有完整的why      │
└─────────────────────────────────────────────────────────┘
                            ↑
┌─────────────────────────────────────────────────────────┐
│                    仲裁层（信号博弈）                        │
│        多Agent信号聚合 → 方向判定 → 仓位建议 → 风险检查      │
└─────────────────────────────────────────────────────────┘
                            ↑
┌─────────────────────────────────────────────────────────┐
│                    研究层（信号生成）                        │
│  ┌─────┐ ┌─────┐ ┌─────┐ ┌─────┐ ┌─────┐ ┌─────┐       │
│  │财报 │ │趋势 │ │资金 │ │估值 │ │地缘 │ │行业 │  ...   │
│  │分析 │ │识别 │ │流向 │ │分析 │ │事件 │ │对比 │       │
│  └─────┘ └─────┘ └─────┘ └─────┘ └─────┘ └─────┘       │
└─────────────────────────────────────────────────────────┘
                            ↑
┌─────────────────────────────────────────────────────────┐
│                    感知层（数据获取）                        │
│      行情数据 · 财报原文 · 新闻资讯 · 产业链数据 · 资金流    │
└─────────────────────────────────────────────────────────┘
```

## 快速开始

### 1. 安装依赖

```bash
pip install -r requirements.txt
```

### 2. 运行示例

```bash
python main.py --stock 000001
```

### 3. 开发新Agent

参考 `docs/AGENT_GUIDE.md`

## 项目结构

```
AIRenaissance/
├── agents/                    # Agent目录
│   ├── __init__.py
│   ├── base.py               # Agent基类
│   ├── signal.py             # 统一信号格式
│   │
│   ├── perception/           # 感知层Agent
│   │   ├── market_data/      # 行情数据
│   │   ├── financial_data/   # 财报数据
│   │   ├── news_data/        # 新闻资讯
│   │   └── fund_flow/        # 资金流向
│   │
│   ├── research/             # 研究层Agent
│   │   ├── financial_report/ # 财报分析
│   │   ├── trend/            # 趋势识别
│   │   ├── valuation/        # 估值分析
│   │   └── ...
│   │
│   ├── risk/                 # 风控层Agent
│   │   └── position/         # 仓位管理
│   │
│   └── cognition/            # 认知层Agent
│       └── reasoning/        # 推理链生成
│
├── arbitration/              # 仲裁层
│   ├── engine.py
│   └── risk_controller.py
│
├── config/                   # 配置文件
│   ├── default.yaml
│   └── agents.yaml
│
├── main.py                   # 入口
├── requirements.txt
└── README.md
```

## 团队协作

- **框架维护**：由核心开发者负责
- **Agent开发**：团队成员认领，在 `agents/` 下创建自己的Agent
- **提交流程**：Fork → 开发 → PR → Code Review → 合并
