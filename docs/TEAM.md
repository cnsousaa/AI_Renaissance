## 协作原则

1. **框架归开发1组** — 其他人不要改 `base.py` 和 `signal.py`
2. **数据归开发3组** — 专家组只调接口，不直接调 API
3. **Skill 可复用** — 财务专家写的 Skill，其他 Agent 也能用
4. **PR 必须 Review** — 任何人提交，至少一人看过再合并
